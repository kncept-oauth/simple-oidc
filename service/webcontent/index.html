Simple OIDC

Simple OIDC provides an instant, flexible auth solution.

What we provide:

For Users

Keep track of which applications and websites have access to your data

Allow for multiple external authorizations - including Google, Facebook and X

Don't want to connect to any other provider? Register using an email or even a phone number directly

For Application Owners

Always able to get a list of your users

Funnel users from multiple sources and providers into one unified solution

Easy to integrate

For Developers

OIDC based flow makes it easy to integrate any application, language or framework

Standards based JWT solution

Open Source, so you can audit the code, and make sense of any extracts you download

Self Hostable, if you _really_ don't want to trust a cloud provider
